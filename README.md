# ✨ Features

- 🌍 **Multi-language** — Supports your system language or custom choice.
- 📸 **Image & video support** — Many formats.
- 🧩 **Grid & mosaic views**
- 🎞️ **Slideshow**
- 🖌️ **Edit & add effects**
- 🔒 **Vault** — Private media.
- ⭐ **Favorites**
- 📂 **Smart categories & albums**
- 🔍 **Advanced search** — Terms & multiple tags.
- 🚫 **Ignore folders/media**
- 🗑️ **Trash bin**
- 📊 **Statistics**
- 🔄 **Update checker**
